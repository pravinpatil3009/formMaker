1. Go to http://jqueryui.com/download and download jquery-ui-1.8.4.custom.zip
   [VERSION: 1.8.4 and COMPONENT: Draggable].

2. Extract the js folder from the zip file to the folder google_virtual_keyboard/js.

3. If you have extracted the contents right, make sure the following files should exists:
    i. [path to modules folder]/google_virtual_keyboard/js/jquery-1.4.2.min.js
   ii. [path to modules folder]/google_virtual_keyboard/js/jquery-ui-1.8.4.custom.min.js
